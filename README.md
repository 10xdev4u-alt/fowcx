# Course Notes: OE22708 Fundamentals of Wireless Communication

## Description

This repository contains a comprehensive set of study materials for the course OE22708: Fundamentals of Wireless Communication. The notes are structured to align with the official course syllabus and are intended for exam preparation.

## Repository Structure

The repository is organized as follows:

-   **Core Files**: Foundational documents that apply to the entire course.
    -   `00_Syllabus.md`: Contains the official course objectives and unit breakdown.
    -   `00_Common_Formulas.md`: A cumulative reference sheet for all mathematical formulas introduced.
    -   `00_Problems_and_Solutions.md`: A collection of solved example problems for practice.

-   **Unit Topics**: The notes are divided into five units, corresponding to the syllabus. Each unit contains:
    -   **Topic Notes**: Individual markdown files for each topic within the unit.
    -   **Assessment File**: A final `_Assessment.md` file with practice questions and detailed answers to test understanding of the unit's concepts.
